## Tenant Smoke Path
1. Login works for each role.
2. Request creation and approval path works.
3. Expense posting works.
4. Vendor invoice flow works.
5. Treasury import and reconciliation path works.
6. Execution health and payout queue surfaces load correctly.
7. Reports center reflects new activity.

## Platform Smoke Path
1. Platform operations hub loads.
2. Production validation summary is visible.
3. Scheduler heartbeat is recent.
4. Failed jobs and queued backlog metrics are visible.
5. Incident history and rollout pages load.
